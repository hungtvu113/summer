### Coding Standards

#### TypeScript

- Lu√¥n s·ª≠ d·ª•ng TypeScript, tr√°nh `any`
- Define interfaces/types r√µ r√†ng
- S·ª≠ d·ª•ng strict mode


#### React Components

- S·ª≠ d·ª•ng functional components
- Prefer named exports
- Props interface b·∫±ng TypeScript

```typescript

  children: React.ReactNode;
  onClick: () => void;
  variant?: 'primary' | 'secondary';

  return (
    <button className={`btn btn-${variant}`} onClick={onClick}>
```

#### Styling

- S·ª≠ d·ª•ng Tailwind CSS
- T·∫°o reusable components cho common patterns
